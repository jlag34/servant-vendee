Servant-Showcase
=================================

Free mobile-optimized catalog for the products on your Servant.

###Set-Up

Create a `.env` file in the root directory and add the following environment variables: 

* PORT
* SESSION_NAME
* SESSION_SECRET
* SERVANT_CLIENT_ID
* SERVANT_SECRET_KEY
* AWS_ACCESS_KEY
* AWS_SECRET_ACCESS_KEY
* AWS_REGION
* AWS_BUCKET
